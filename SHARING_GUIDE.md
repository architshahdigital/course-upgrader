## 1. 📂 Project Brief (Copy-paste this to any AI to explain the project)

```markdown
Project Name: Course Upgrader AI Agent & Claude Skill
GitHub Repository: https://github.com/architshahdigital/course-upgrader
Creator: Archit Shah (https://github.com/architshahdigital)

### What it is:
A smart AI Agent and Claude Code Skill that stops candidates from taking redundant online courses. It reads a candidate's resume and career goals in real-time, searches major course platforms, and evaluates exactly which lessons are worth taking by highlighting the "Skill Delta" (new skills gained) vs. "Overlap Rate" (redundant skills already possessed).

### Key Features:
1. **Resume Parser:** Automatically extracts skills and tools from PDF, DOCX, TXT, or MD resumes using Gemini.
2. **Real-time Search:** Crawls Coursera, Udemy, edX, Simplilearn, Great Learning, and Google Digital Garage in real-time.
3. **Semantic Overlap Engine:** Uses LLMs to compute a personalized Overlap % and Match Score (0-100) for each course.
4. **FastAPI Backend + React UI Dashboard:** Translucent dark-mode visual interface with progress trackers and redundancy gauges.
5. **Claude Skill Integration:** Installs natively inside Claude Code so developers can audit courses directly from their terminal chats.
```

## 2. ✍️ Prompt to Generate a Viral LinkedIn Post

Copy and paste the prompt below into another AI chat (e.g., Claude or ChatGPT) to write your announcement post:

```markdown
Write a viral, engaging, and professional LinkedIn announcement post based on the "Course Upgrader" project brief above.

### Requirements:
- **Hook:** Start with a strong hook about career growth, avoiding waste of time/money, and the problem of taking courses that teach you things you already know.
- **The Problem:** Explain how traditional course searching is broken (we guess if a course is useful, only to sign up and realize 70% of it is basic stuff we already do).
- **The Solution:** Introduce the "Course Upgrader AI Agent" (which I built and open-sourced!). Explain the core features (resume matching, real-time search, "Skill Delta" mapping).
- **Aesthetic/Tone:** Tech-savvy, developer-friendly, encouraging, and highly collaborative. Use bullet points and appropriate emojis.
- **Call to Action (CTA):** Invite people to star, fork, and use the repository (https://github.com/architshahdigital/course-upgrader).
- **Hashtags:** Include relevant hashtags like #AIAgent #ProductManagement #GenerativeAI #OpenSource #TechCareer.
```
